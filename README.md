# TANet Roamer
https://chrome.google.com/webstore/detail/tanet-roamer/oholmaopjilhledjkmfpolbbjoacmcak?hl=zh-TW

## 宗旨
以最方便的操作流程，取得全台灣的 TANetRoaming WIFI 的漫遊認證。

## 使用說明
一鍵登入校園 WI-FI 的Chrome APP，目前測試中。

## 特色
 - 記憶帳號密碼，存於Google帳戶，可跨裝置。
 - 一鍵登入校園 WI-FI ，方便快速。

## 適用學校
 - 國立彰化師範大學
 - 國立中興大學
 - (其他校園待測試)
